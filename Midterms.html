    Cascading Style Sheets

    In 1994, CSS was invented by Hakon Wium Lie. Versions include CSS 1, CSS 2, CSS 2.1, and CSS 3.

    Cascading Style Sheets is used for styling the layout of webpages specifically: HTML, XML, XHTML, etc.It represents how HTML elements are displayed on screen externally or internally. The use of CSS is for the preference of users in defining their own style of web pages that includes the design, layout, and other view perspective in other devices and screen sizes.
