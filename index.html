    Uniform Resource Locator (URL)

    A Unifrom Resource Locator (URL)refers to a web resource that specifies the location on the computer network. A URL is a specific category of a Uniform Resource Identifier (URI). URL's are used commonly to reference web pages such as HTTP, FTP, Mail to JDBC and many other applications. Most of the web browsers display URL's of a web page in the top part of the page in an adress bar. An example of which is the from https://www.webtech.com/index.html which indicates the protocol (http/https), the hostname (www.webtech.com), and the filename (index.html).
